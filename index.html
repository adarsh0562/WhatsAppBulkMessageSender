WhatsApp Bulk Message Sender

Automation Testing | Cucumber | TestNG | Selenium | Java | Extent Report

📌 Features

  • Automated WhatsApp message sending using Selenium
  • Data-driven from Excel for contacts/messages
  • Reusable TestNG framework with Page Object Model
  • Beautiful Extent Report & Allure integration

🚀 Technologies Used

  • Java
  • Selenium WebDriver
  • TestNG + Cucumber
  • Extent Reports
  • Apache POI (for Excel)
  • Allure
  • Git & Jenkins CI

⚙️ How to Use

  1. Clone the repo
  2. Run `mvn clean test`
  3. View the generated report

📤 Jenkins Integration

Jenkins is configured to run this project automatically and push the updated report/testng/index.html to the GitHub repo. You can always view the latest report by clicking the button above.
